## 📌 Project Description

This project analyzes an **NBA players dataset across multiple seasons** to uncover performance trends and player statistics over time.

Using Python, the dataset was cleaned and explored to examine key metrics such as points, assists, rebounds, player efficiency, and physical attributes. The analysis focuses on identifying patterns in player performance, comparing statistics across seasons, and understanding relationships between different variables.

This project demonstrates practical skills in data preprocessing, exploratory data analysis (EDA), and data visualization applied to a real-world sports dataset.

---

## 🎯 Key Objectives

* Analyze player performance across different NBA seasons
* Explore relationships between statistics (points, assists, rebounds, etc.)
* Identify trends and patterns in player data
* Perform data cleaning and preprocessing
* Visualize insights using graphs

---

## 📊 Key Insights

* Player performance varies significantly across seasons and teams
* Strong relationships exist between usage rate and scoring performance
* Physical attributes (height, weight) influence certain performance metrics
* Some players maintain consistent performance, while others show fluctuations over time

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib / Seaborn
* Jupyter Notebook

---

## 📂 Dataset Features

* Player name, team, and season
* Points, assists, rebounds
* Player height and weight
* Shooting efficiency metrics (TS%)
* Advanced stats (usage rate, net rating, etc.)

---

## 📸 Visualizations

* Distribution of player performance (histograms)
* Comparison between statistics (scatter plots)
* Seasonal trends (line charts)
* Team/player comparisons (bar charts)

---

## 🎯 Conclusion

The analysis provides valuable insights into player performance trends and highlights how different factors influence success in professional basketball. This project reflects the use of data analytics techniques to extract meaningful information from sports data.
